<내용>
본 프로젝트는 스프링부트로 작성된 XML과 JSON 사이의 양방향 변환 REST API임

<사용된 오픈소스>
1. Jackson Databind	
-웹사이트:https://mvnrepository.com/artifact/com.fasterxml.jackson.core/jackson-databind   
-GPL:Apache
-사용목적:JSON 유효성 검사

2. Ehcache
-웹사이트:https://mvnrepository.com/artifact/net.sf.ehcache/ehcache 
-GPL:Apache 2.0
-사용목적:변환 통계 데이터의 유지 및 변경

3. Thymeleaf
-웹사이트:https://www.thymeleaf.org/ 
-GPL:Apache 2.0
-사용목적:웹페이지 구현

4. Bootstrap
-웹사이트:https://getbootstrap.com/ 
-GPL:MIT
-사용목적:웹페이지 디자인

5. Chart.js
-웹사이트:http://www.chartjs.org/ 
-GPL:MIT 
-사용목적:변환 통계 데이터의 차트 시각화
